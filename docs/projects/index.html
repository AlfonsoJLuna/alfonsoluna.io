Projects

Here is a selection of some of my latest projects, most of them are open-source:

Electronics

  • openhandheld-mx — An open-source STM32 development board in the shape of a handheld game console.
  • chip8stm32 — STM32 based CHIP-8 and Super-CHIP handheld emulator device.

Emulation

  • nesswemu — A NES emulator in an early stage of development. Currently only the MOS 6502 interpreter is implemented.
  • chip8swemu — Multiplatform CHIP-8 and Super-CHIP emulator written in C. Includes a Windows/Linux compatible frontend.

Others

  • minesweeper — A clone of the classic Minesweeper game written in C using SDL2.

More projects can be found on my GitHub.
